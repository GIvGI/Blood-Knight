# Blood-Knight
Survivor-like game I made using LOVE2D. 
It is a very quick and small game that I made while learning LOVE2D and game development. Since it was made in a learning process, some things are done unconventionally. I might fully finish it and add/remake some stuff or I might start making a new game similar to this.
